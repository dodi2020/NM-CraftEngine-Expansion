# NexoMaker-CraftEngine-Expansion

The **NexoMaker-CraftEngine-Expansion** is designed to allow the creation of custom items within **NexoMaker** for use with the **CraftEngine** mod/framework.

---

## ⚠️ **Alpha Stage Warning** ⚠️

**Please note that this project is currently in an early Alpha stage.**

This means:
1.  **It is not yet complete or fully functional.** Features may be missing, broken, or change without notice.
2.  **It is not suitable for productive or critical use.** Use it only for testing, development, and non-essential projects.

We appreciate your patience as we work toward a stable release!

---

## Current Development Version

The most up-to-date version of the source code is actively being developed and can be found in the **`develop`** branch of this repository.

If you are looking to test the latest features or see recent changes, please check out and clone the `develop` branch.
